fluff REST API
==============

All API urls must be preceded by '[admin path]/api'.

- GET /auth
- GET /auth/:id
- POST /auth
- PUT /auth/:id
- DELETE /auth/:id

- GET /users
- GET /users/:id
- POST /users
- PUT /users/:id
- PUT /verify/:token
- DELETE /users/:id

- GET /sites
- GET /sites/:id
- PUT /sites/:id

- GET /pages
- GET /pages/:id
- POST /pages
- PUT /pages/:id
- DELETE /pages/:id

- GET /views
- GET /views/:id
- POST /views
- PUT /views/:id
- DELETE /views/:id

- GET /vars
- GET /vars/:id
- POST /vars
- PUT /vars/:id
- DELETE /vars/:id
